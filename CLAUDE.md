# CLAUDE.md — media-download

## Overview
Downloads all media files (images, videos, subtitles) from any web page into an organized folder schema. Includes a secondary tool for downloading EPUB publications.

## Tech Stack
- Python
- Docker (multi-image: `Dockerfile-media` for media, `Dockerfile-epubs` for publications)
- pytest (test suite in `tests/`)
- GitHub Actions CI

## Development
```bash
pip install -r requirements.txt
python src/media-vtt.py        # Media/subtitle downloader
python src/publications-epub.py # EPUB publication downloader

# Tests
pip install -r requirements-test.txt
pytest tests/
```

## Architecture
- `src/` — Main scripts (`media-vtt.py`, `publications-epub.py`)
- `tests/` — pytest test suite
- `docs/` — Documentation and images
- `Dockerfile-media` — Container for media downloading
- `Dockerfile-epubs` — Container for EPUB downloading

## Key Rules
- Never hardcode credentials or authentication tokens
- Docker images use semver tags on Docker Hub (`drumsergio/media-download`)
- Two separate Dockerfiles serve different use cases; do not merge them
- Maintain test coverage (Codecov integrated)
